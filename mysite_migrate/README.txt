
----------------------------------
Dynamic Mysite migrate Module
----------------------------------


CONTENTS OF THIS FILE
---------------------

 - This module is used to migrate(import) data from CSV to drupal using migrate module.

Drupal functionality used
-------------------
  - user field migration
  - profile2 field migration
  - CSV to drupal data import
  - variables API to store variables
  - Password reset.(user forced to reset password after first successful login)
  - db_update
  - drupal_get_path
  - HOOKS
    - hook_migrate_api()
